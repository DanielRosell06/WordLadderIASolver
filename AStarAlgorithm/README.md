# Algoritmo A* WordLadder

## graph_conversor.c + grafo.h

Converte o grafo gerado (.txt) em um array bidimensional legível em C, com o objetivo de tornar a busca em O(1). O arquivo de resultado é o grafo.h 

> grafo.h é incluso no gitignore por questão de tamanho.

## Compile

Command to compile:

gcc main.c node.c operations.c database.c -o main
